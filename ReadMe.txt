[使用说明]

打开方式:
1>.双击IShow.exe,通过选择文件打开图片
2>.支持资源管理器中选中一张或者多张(限制一次最多50张)图片,
   通过右键菜单点击"打开",打开一张或者多张图片.
3>.支持双击图片文件的方式打开.

操作控制:支持打开的图片上右键菜单和快捷键两种方式,以下为快捷键说明

鼠标右键按下          : 弹出菜单
鼠标左键拖动          : 移动图片
鼠标滚轮向上滚动      : (地图式)放大
鼠标滚轮向下滚动      : (地图式)缩小
鼠标双击/ESC键        : 关闭
向上箭头              : 切换到当前图片的上一张
向下箭头              : 切换到当前图片的下一张
向左箭头              : 逆时针旋转90°(仅适用于静态图片)
向左箭头              : 顺时针旋转90°(仅适用于静态图片)
Ctrl+鼠标滚轮向上滚动 : 顺时针旋转(用于静态图片)
Ctrl+鼠标滚轮向下滚动 : 逆时针旋转(用于静态图片)
鼠标空格键(SPACE)     : 暂停/继续(仅适用于GIF图片)
Ctrl+鼠标滚轮向上滚动 : GIF图片帧率加快一倍(用于GIF图片)
Ctrl+鼠标滚轮向下滚动 : GIF图片帧率减慢一倍(用于GIF图片)

注意:
1>.以上全部鼠标操作均是在图片区域内.
2>.支持中文和英文,根据系统当前语言自动显示.
